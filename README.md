Aetheria AI é uma plataforma de inteligência documental de próxima geração, desenvolvida para transformar arquivos brutos em insights estruturados através de uma interface de alta performance. O projeto foi concebido sob a estética Liquid Glass (Apple OS 26 Style), unindo o poder do Gemini 2.5 Flash a uma experiência de usuário fluida e refrativa.

Motor Cognitivo de Ponta: Integração nativa com a API Google GenAI (Gemini 2.5/3.1), permitindo análises contextuais profundas em milissegundos.

Interface Liquid Glass: UI customizada via CSS Injection, apresentando efeitos de transparência extrema, backdrop blur dinâmico e bordas neon verde lima.

Fluxos de Ação Rápida: Botões pré-configurados para Resumo Executivo, Extração de Insights e Auditoria de Erros Lógicos.

Exportação Inteligente: Sistema de geração de relatórios em tempo real com download instantâneo dos resultados processados.

Arquitetura Resiliente: Tratamento avançado de Rate Limiting (Error 429) e roteamento dinâmico de modelos de API.


Stack Tecnológica
Linguagem: Python 3.11+
IA Engine: Google Gemini 2.5 Flash
Framework: Streamlit
Interface: Custom CSS 
Ambiente: Linux (Endless OS / Debian) com isolamento via venv
